# YieldCurveInterpolator

Библиотека для моделирования рыночных кривых депозитных ставок на основе методов снижения размерности с устойчивостью к пропускам в данных.

Основной сценарий использования: восстановление полной кривой ставок по аукционным данным, где часть бакетов может отсутствовать в отдельные даты, а сделки имеют разный объём и требуют взвешивания.

## Для кого

- **Quant-аналитики**, работающие с кривыми процентных ставок.
- **Трейдеры денежного рынка**, которым нужна оценка справедливой ставки для выявления арбитражных возможностей.
- **Риск-менеджеры**, которым требуется мониторинг структурных сдвигов в кривой.
- **Исследователи**, изучающие факторную структуру доходности (Level, Slope, Curvature по Litterman-Scheinkman).

## Возможности

- Взвешенная агрегация ставок по объёму сделок (`amount`).
- Устойчивая интерполяция пропусков через EM-PCA.
- Расширяемая архитектура движков (EM-PCA, PPCA, Kalman).
- Внутренняя диагностика: сходимость, объяснённая дисперсия, нормальность остатков, автокорреляция, выбросы.
- Внешние метрики качества: RMSE, MAE, directional accuracy, косинусное сходство формы, экономические метрики (PnL).
- Интерактивная визуализация через Plotly с экспортом в HTML.
- Бэктест торговых стратегий с учётом duration, транзакционных издержек, slippage и funding cost.
- Промышленные риск-метрики: Sharpe, Sortino, Calmar, Max Drawdown.
